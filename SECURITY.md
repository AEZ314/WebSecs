# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, report it privately to:

**efe@efezaladin.dev**

Do not create a public issue.

Please include:

- Description of the issue
- Steps to reproduce
- Affected version or commit
- Potential impact
- Any proof of concept

## Disclosure

Please allow time for investigation and remediation before public disclosure.

## Supported Versions

We only support the latest version of this project for security fixes.
